# VibeStack

VibeStack is a curated intelligence dashboard for AI-native developers. It helps developers answer the question: "What should I add to my AI stack this week?"

## Tech Stack
- React 19
- Vite
- TypeScript
- Tailwind CSS v4
- React Router DOM
- Lucide React (Icons)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Architecture

- `src/components/`: Reusable UI components (Card, Badge, FilterBar, Pagination, Layout)
- `src/pages/`: Page components (Dashboard, About)
- `src/hooks/`: Custom React hooks (`useUpdates` for data fetching)
- `src/lib/`: Utilities and mock data

## Design System
- **Background**: `#0B0F14`
- **Accent**: Warm amber/gold (`#D4A373`)
- **Typography**: Inter (Sans), JetBrains Mono (Monospace)
- **Aesthetic**: Minimal, sharp, confident, calm. Stripe + Linear + Homebrew inspired.
